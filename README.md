### Features
- :gift: **Rich Integrations and Out of the box**
- :art: **Highly Customizable**
- :rainbow: **Support `Vue`/`Angular`/`React`**
- :rocket: **Available reporting in `Web Worker` to avoid block the main thread**

### Quick Start
```ts
import { init } from "@booji/browser"

init({
  dsn: "__DSN__",
  appKey: "__KEY__"
})
```
